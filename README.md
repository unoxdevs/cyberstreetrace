# 🏁 Cyber Street Race

Connecting the world of cyber racing with the real world.

## 📦 Install Repository

```bash
git clone https://github.com/unoxdevs/cyberstreetrace
cd cyberstreetrace
npm install
```

## 🚀 Quick Start

```bash
npm run dev
```
